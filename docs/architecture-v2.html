① 增值: OA 端 USER 取檔通道+ 主管圖沒涵蓋

主管圖下載端是「正式環境系統」, 只給機器拉。本方案補一條「給人用」的下載通道, 3 種選法依需求:
🖱️ SMB 掛載拖拉 (推薦)
檔案總管打 \\SF\architecture\ Ctrl+A 拖到本機, 最快
📦 Portal 批次 ZIP
瀏覽器勾多檔 → 打包成 ZIP, 不會掛 SMB 的人用
📥 Portal 逐個下載
瀏覽器點檔案 [下載], 適合單檔

② 增值: 業務 ANY 制簽核 + AD 群組自助維護+ 補純技術 ACL 缺口

主管圖純技術 ACL, 沒擋「上傳錯檔 / 外流敏感資料」風險。本方案加業務分權:
📝 5 人 ANY 制
任 1 同意即放行 / 任 1 駁回即駁回, 不卡業務
📋 批次簽核
同 u0X 5 分鐘內檔案聚合, 1 行 1 批, 可展開細選
👥 AD 群組自助維護
業務負責人在 Portal「我的群組」自己加減成員, IT 不介入
⏰ 7 天保留 + 超時駁回
home 7 天 + samba 7 天, 超時視同放棄

③ 增值: Portal 集中視覺化+ SIEM 上線前的橋接

主管圖 Log 直接丟 SIEM, 但 SIEM 上線前 IT 怎麼查? 本方案 Portal 自帶:
🔍 全公司稽核查詢
多條件查 + CSV 匯出 + GPT 分析 (去識別化)
🔗 事件鏈追蹤
PAM → AP → SF → SMB 一頁顯示 (階段二)
💚 系統健康儀表板
服務狀態 + 排程 + 容量, 不用 RDP
📜 系統 Log 集中頁
Portal/IIS/OpenSSH/SMB/SQL/Scheduled 6 來源切換
📦 Debug Bundle 收集
一鍵打包 (去識別化) → 給 GPT Enterprise 分析
🚀 SIEM 漸進整合
AuditLog 預留 CEF/syslog 欄位, 階段二一個 forwarder 接上
