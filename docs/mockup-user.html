📦 批次打包下載 — 流程說明
  1. 勾選您要下載的檔案 (上方表格 checkbox)
  2. 點「📦 打包成 ZIP 下載」
  3. 系統後台用 svc_portal 串流讀取 samba 檔案 → 即時打包 (不存中間檔)
  4. 瀏覽器收到 sf_u01_20260518_xxxx.zip
  5. AuditLog 記錄: BATCH_ZIP_DOWNLOAD · 您的 AD 帳號 · 包含哪些檔
  6. 大檔注意: ZIP 不存中間檔, 大檔 (>1GB) 建議改用 SMB 拖拉
ℹ️ 您只看得到「您 AD 群組有權的」samba 目錄。其他部門 (HR / 財務 / 資安) 您看不到。
ℹ️ samba 上的檔案 7 天後自動清空。請及時下載或請放行人重新處理。
