### 2. errorhandler(500) 改 inline HTML
之前 500 handler 跑 `render_template('error.html')` — 萬一 error.html 又依賴 base.html, base.html 又 fail (例如 url_for 找不到 endpoint), 形成 template loop, 最終回 generic 500 沒任何資訊.

新版 try render template, 失敗 fallback **純 inline HTML** (不依賴任何 template):
```html
<h1 style="font-size:60px;color:#c00;">500</h1>
<h2>Internal server error</h2>
<p>請聯絡 IT 管理員, 或回到 <a href="/">首頁</a>.</p>
```

DEBUG 模式下還會印完整 traceback (escape 後), 方便診斷.

- `log_audit` 永遠包 try (寫不到 DB 不能擋業務)
