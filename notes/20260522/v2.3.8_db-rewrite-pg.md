# Note 2026-05-22 v2.3.8 — db.py 重寫 MSSQL→PostgreSQL + 補 python-dotenv

## 變更

### 1. portal/app/db.py 整檔重寫
- pyodbc → psycopg2
- 內建 `_translate_sql()`:
  - `?` placeholder → `%s`
  - `SYSUTCDATETIME()` → `(NOW() AT TIME ZONE 'UTC')`
- execute_returning_id 自動加 `RETURNING id` (取代 SCOPE_IDENTITY)
- caller 不用改 (admin.py / approval.py / audit_helper.py / auth.py 仍用 `?` 跟 SYSUTCDATETIME)
- schema 用 INTEGER 0/1 當 boolean (跟 MSSQL 行為一致), 不用 PG 原生 BOOLEAN

### 2. 新增 PyPI wheel: python-dotenv
- URL: https://files.pythonhosted.org/packages/6a/3e/b68c118422ec867fa7ab88444e1274aa40681c606d59ac27de5a5588f082/python_dotenv-1.0.1-py3-none-any.whl
- 大小: 約 20 KB
- 用途: config.py 讀 .env 設定

### 3. install_portal_all_in_one v2.3.8
- REQUIRED_WHEELS 加 python_dotenv-*.whl (從 3 個變 4 個)
- Step 4 unzip 清舊 dotenv/python_dotenv site-packages 污染
- import 驗證加 dotenv

- 包含新版 db.py
