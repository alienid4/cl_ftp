# Note 2026-05-22 v2.4.5 — UI 對齊 mockup + 修 /admin/biz 500 + Linux health

## 情境

USER 用 dev-mode 進 portal 後抱怨「這跟規劃的畫面差很多」.
比對 `docs/mockup-admin.html` / `docs/mockup-user.html` 跟實際 portal 跑出來的差異:

| 項 | mockup 版 | v2.4.4 實際 |
|---|---|---|
| navbar | 國泰綠 + admin 橘色 dot | 純文字連結 |
| admin 警示帶 | 橘色橫條「您目前以管理員身分登入」 | 沒 |
| KPI 卡 | 國泰風 6 卡 (綠/橘/紅顏色) | 沒 |
| 表格 | data-table CSS (header bar, hover) | 純 inline style 框線 |
| 業務代號清單 | 完整 mockup 樣式 | 簡易 placeholder |
| 系統健康 | health-grid 圖示 + 卡片 | "No such file: powershell.exe" |
| 字型 + 卡片 | cathay.css 全套 | 都沒 |

### 根因

1. CSS 沒拷到 `portal/app/static/css/` — Flask `url_for('static', ...)` 載不到, 整頁裸文字
2. Templates 用「最精簡 placeholder」沒套 mockup CSS class
3. admin.py health() 呼叫 `powershell.exe` — Linux 沒有
4. /admin/biz 500 — 沒 try/except, DB / template 任一個錯就整頁炸

## 修法

`docs/static/css/{cathay,admin,mockup-extra}.css` → `portal/app/static/css/`
total 約 41 KB, 含完整國泰風格 layout / nav / cards / table / KPI / health-grid / badges 等.

### 2. Templates 重寫 (4 個)

#### base.html
- navbar 用 `.navbar-brand .dot` + `.nav-links` 樣式
- admin 身分時 dot 變橘色, 顯「· 管理員」, 加 admin warning bar

#### home.html
- 分支 admin / user 兩種版型
- admin: 6 個 KPI 卡 (啟用業務代號 / 今日上傳 / 本月簽核 / 本月下載 / 今日 DENIED / 待簽批次) + 健康速覽
- user: 4 個 KPI 卡 (待簽 / 已同意 / 已駁回 / 我所屬群組)
- 共用: 待簽批次表格 (前 5 筆 + 跳轉完整列表)

#### admin_biz_list.html
- 對齊 mockup biz-codes 表
- 代號 / 業務名稱 / 負責人 / 簽核群組 / samba 目錄 / 狀態 / 操作
- stop 用 inline style, 改用 `.data-table` / `.code-inline` / `.badge` class

#### admin_health.html
- 對齊 mockup health-grid
- 服務狀態 (sf-portal / nginx / postgresql / firewalld) ✅⚠️❌ 圖示
- 主機資源 KPI (CPU / Memory / 磁碟 / Uptime), 動態變色
- DB 統計 (AuditLog / BusinessCode / Batch 筆數)
- 錯誤 fallback

### 3. admin.py health() 改 Linux

從:
```python
result = subprocess.run(['powershell.exe', '-File', r'C:\...\health_check.ps1', '-Json'], ...)
```

改成:
- `systemctl is-active sf-portal/nginx/postgresql/firewalld` 撈服務狀態
- `/proc/loadavg` 撈 CPU load → 推估 cpu_pct
- `/proc/meminfo` 撈 mem_pct
- `shutil.disk_usage('/')` 撈 disk_pct
- `/proc/uptime` 撈 uptime
- `psql` 撈 AuditLog / BusinessCode / Batch 筆數

完全 Linux 原生, 不依賴 PowerShell.

### 4. main.py home() 加 KPI + try/except

每個 DB query 用 `_safe(lambda: ...)` 包住, 任一 query 失敗回 0, 整頁不 500.

### 5. admin.py biz_list 加 try/except

`get_business_codes()` 失敗 → flash 訊息 + render 空 codes, 不 500.
