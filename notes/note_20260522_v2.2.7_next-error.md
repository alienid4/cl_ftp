## 常見後續錯誤

| 錯誤訊息 | 原因 | 修法 |
|---|---|---|
| `ModuleNotFoundError: No module named 'XXX'` | 又缺套件 | 再從 PyPI 抓 wheel 放 /tmp/ftp-lab/ |
| `psycopg2.OperationalError: could not connect to server` | postgresql 沒跑 / port 5432 沒監聽 | `systemctl status postgresql` |
| `psycopg2.OperationalError: FATAL: password authentication failed` | portal user 密碼跟 appsettings.json 不一致 | 重設 portal password |
| `FileNotFoundError: [Errno 2] No such file or directory: '/data/exchange'` | DATA_ROOT 目錄沒建 | `mkdir -p /data/exchange && chown nginx /data/exchange` |
| `PermissionError: [Errno 13]: '/opt/portal/logs/...'` | logs 目錄 owner 不對 | `chown -R nginx:nginx /opt/portal` |
| `AttributeError: module 'X' has no attribute 'Y'` | 套件版本不對 | 看是哪個套件, 補裝對應版本 |
| `RuntimeError: A secret key is required` | SECRET_KEY 沒設 | 看 appsettings.json |
